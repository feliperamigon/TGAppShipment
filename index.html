Pregunta Generadora

¿Cómo facilitar el envío de mercancías mediante un aplicativo multiplataforma, basado en modelos de economía colaborativa?

Objetivos

¿ A que se quiere llegar realmente?

Objetivo General

Construir una aplicación multiplataforma basada en economías colaborativas que permitan a las personas encontrar una alternativa con base tecnológica para el envió de mercancías a nivel nacional.

Objetivos Específicos

  • Realizar proceso de ingeniería de Requerimientos para construir el aplicativo Appshipment
  • Diseñar un aplicativo multiplataforma con base en tecnologías web y móvil
  • Implementar el aplicativo AppShipment basado en el diseño realizado anteriormente
  • Realizar procesos de validación y pruebas orientadas a TDD

Historia

Conoce el proceso de desarrollo

AppShipment es la aplicación resultante de este trabajo de grado, la cual busca fomentar prestaciones de servicios basados en la confianza entre usuarios y economías colaborativas a la hora de enviar mercancías, donde a través de una plataforma las personas, usuarios prestadores del servicio, pueden pueda poner a disposición su tiempo y recursos para transportar mercancías de otras personas, usuarios demandantes del servicio, y obtener remuneración económica.
